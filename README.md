# Office-flash 365 — Site vitrine

Site vitrine moderne pour un expert formateur Microsoft Office 365 basé à Bingerville, District d'Abidjan (Côte d'Ivoire).

## Stack
- HTML5 + Tailwind CSS (via CDN)
- Icônes Lucide
- Typographies Google Fonts (Plus Jakarta Sans + Inter)
- JavaScript vanilla pour interactions

## Palette
- **Bleu profond** `#0B2A4A` — professionnalisme
- **Vert vif** `#22C55E` — croissance / tech
- **Orange dynamique** `#F97316` — énergie / action

## Structure
- Header (logo, menu, CTA "Devis Gratuit")
- Hero avec photo formateur & icônes Word/Excel/PowerPoint flottantes
- Preuve sociale (logos structures locales)
- Formations : Excel Pro, Word & PowerPoint, Collaboration Cloud
- Coaching & localisation Bingerville (carte OSM)
- À propos / pourquoi nous
- CTA final (Appel, WhatsApp, Email)
- Footer

## Lancer en local
Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
npx serve .
```

## Personnalisation
- Remplacer l'image du hero dans `index.html` (tag `<img src=...>`)
- Mettre à jour numéro WhatsApp / tel / email dans la section CTA et le footer
- Modifier les logos clients (section "Ils nous font confiance")
